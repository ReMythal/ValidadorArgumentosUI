Limitaciones del sistema
✗ Condicional -> con expresiones complejas — Solo funciona entre variables simples. (p && q) -> r no funciona, usa p -> r
✗ Bicondicional <-> con expresiones complejas — Solo funciona entre variables simples. (p && q) <-> r no funciona, usa p <-> r
✗ Negación antes de condicional — !p -> q se interpreta como !(p -> q) en vez de (!p) -> q . Puede dar resultados incorrectos.
✗ Implicaciones / dobles implicaciones encadenadas o mezcladas — p -> q -> r , p <-> q <-> r o p -> q <-> r no funcionan. Sepáralas en premisas individuales.
✗ Variables de una sola letra — Solo usa p, q, r, s... No uses nombres largos como rain (se interpreta como 4 variables: r, a, i, n).
✓ Formatos que SÍ funcionan:
p && q p || q p -> q p <-> q !p !p && q p || !q
